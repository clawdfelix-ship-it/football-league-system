# 🏆 足球聯賽管理系統

專業的足球聯賽管理平台，支持積分榜計算、賽程管理和比賽結果記錄。

## 🚀 快速部署

1. 登入 [Vercel](https://vercel.com)
2. 點擊 "New Project"
3. 導入 GitHub 倉庫
4. 設定環境變量：
   - `NEXTAUTH_URL`: 你的應用網址
   - `NEXTAUTH_SECRET`: 隨機密鑰
5. 點擊 "Deploy"

## 🔑 預設帳號
- 管理員: `admin@football.com` / `password`
- 用戶: `user@football.com` / `password`

## ✨ 功能特色
- ⚽ 即時積分榜
- 📊 賽程管理
